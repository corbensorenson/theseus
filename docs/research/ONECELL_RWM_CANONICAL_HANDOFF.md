# 17. Anytime Evidence-Guided Search

The search frontier is ordered by expected value of computation:

[
\operatorname{VOC}(a)
=====================

\mathbb E[\Delta V_{\text{goal}}\mid a]
+
\lambda_i I_{\text{decision}}(a)
--------------------------------

## C_{\text{compute}}(a)

## C_{\text{latency}}(a)

## C_{\text{risk}}(a)

C_{\text{privacy}}(a)
]

Search pseudocode:

```python
frontier = PriorityQueue()
frontier.push(initial_state)

while frontier and budget.remaining():
    state = frontier.pop_max_voc()

    working_view = vcm.compile_view(
        state=state,
        protected_obligations=True,
    )

    proposals = kernel.propose(
        view=working_view,
        budget=budget,
    )

    for proposal in legalize_and_mask(proposals, state):
        successor, receipt = runtime.execute_proposal(
            state=state,
            proposal=proposal,
        )

        assessment = verifier.assess(
            predecessor=state,
            successor=successor,
            receipt=receipt,
        )

        if assessment.complete_and_committable:
            return commit(successor, receipt)

        if assessment.promising:
            frontier.push(successor)

        if assessment.requires_memory_fault:
            frontier.push(
                resolve_memory_fault(successor, assessment)
            )

return typed_abstention(frontier, residuals=True)
```

Search must remain interruptible and return the best supported partial state when budgets expire.

## APPLY

* an exact primitive;

* a world model;
* or a simulator.

## EXPLORE

Create or schedule:

* alternative hypotheses;
* plans;
* program candidates;
* counterfactuals;
* or search branches.

## CHECK

Test:

* a proposition;
* a constraint;
* a program;
* a prediction;
* an artifact;
* a memory dependency;
* or a proposed state transition.

## COMMIT

Stage a durable:

* belief update;
* artifact;
* plan;
* skill candidate;

## STOP

Return:

* success;
* partial result;
* abstention;
* escalation;
* suspension;
* or typed failure.

These six operations are the control interface. They do not replace exact query and program languages.

A semantic solver without a world model has a limited ceiling.

```text
raw observation

→ grounded object/event

→ world hypothesis
→ prediction
→ observed consequence

```

[

at several scales:

* immediate event transition;

It should support:

* continuous latent state;

* uncertainty;
* multiple particles or hypotheses;
* intervention-conditioned prediction;

## Semantic promotion rule

A neat cluster is not yet a concept.

A provisional representation becomes a durable concept only when it demonstrates utility in one or more of:

* prediction;
* control;
* retrieval;
* compression;
* explanation;
* transfer;
* or skill composition.

Concept identity must remain versioned and revisable.

# 22. Memory hierarchy

OneCell sees a unified memory API, but Theseus should distinguish four functions.

## 1. Workspace memory

* very small;
* active;
* differentiable;
* temporary;
* bounded by slots.

## 2. Exact artifact memory

* durable;
* content-addressed;
* typed;
* versioned;
* provenance-bearing;
* transactionally updated.

This is authoritative within its recorded scope.

## 3. Adaptive neural memory

* fast-changing;
* approximate;
* disposable;
* useful for recent associations and local dynamics.

It may suggest. It may not silently establish fact or authority.

## 4. Skill/model library

* reusable;
* executable;
* contract-bound;
* qualified;
* monitored;
* and replaceable.

The recurrent cell itself should store general cognitive priors, not the complete encyclopedia or procedural library.

# 23. Verification model

Every neural output is a proposal.

Verification levels:

| Level | Mechanism                                            | Authority             |
| ----- | ---------------------------------------------------- | --------------------- |
| 0     | type, schema, permission, integrity, resource check  | hard veto             |
| 1     | exact execution, oracle, unit test, proof checker    | scoped acceptance     |

| 3     | learned critic or model ensemble                     | advisory              |

A learned model must not be the sole verifier for durable skill promotion.

# 30. Learning timescales

OneCell needs three learning timescales.

## Immediate recurrence

Milliseconds to minutes.

* no durable weight changes;
* active reasoning;
* search;
* planning;
* execution.

## Fast advisory adaptation

Minutes to days.

* adaptive neural memory;
* temporary local models;
* task-specific statistics;
* disposable heuristics.

These may influence proposals but cannot establish durable facts.

## Quiescent consolidation

Controlled intervals.

1. stop online mutation;
2. freeze an evidence snapshot;
3. replay successes and failures;
4. revise skills and concepts;
5. train candidate neural updates;
6. run full regression and transfer suites;
7. shadow the candidate;
8. promote or roll back through SCF.

The core is stable online but not permanently frozen.

# 37. Falsification criteria

OneCell should be rejected or substantially revised if:

* recurrence fails to generalize beyond trained depth;
* pointer memory remains unreliable;
* exact state is repeatedly corrupted by latent updates;
* explicit search costs exceed its benefit;
* query routing fails to find critical dependencies;
* skill compilation produces near-duplicate memorization;
* skill-routing cost grows faster than reuse value;
* frozen-core capability does not increase;
* the world model adds no planning advantage;
* shared weights create unacceptable interference;
* total task cost loses consistently to matched Transformers;
* apparent gains disappear without a large language adapter;
* or gains depend on privileged deterministic tooling.

Negative results must remain in residual escrow and the experiment ledger.

A core claim about cumulative intelligence should be tested by holding neural parameters fixed while external skills and representations accumulate.

A governed system must preserve the difference between evidence, interpretation, belief, prediction, and reality.

## I. Residual-aware KISS

Architectural simplicity must include displaced search, memory, verification, repair, and maintenance burdens.

## J. Theseus–OneCell synthesis

Theseus can serve as the evolutionary operating system within which alternative cognitive substrates compete without replacing governance, memory, or evidence infrastructure.
